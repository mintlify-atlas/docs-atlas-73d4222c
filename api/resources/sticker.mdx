Stickers are expressive images that can be sent in messages. This resource provides endpoints to manage guild stickers and retrieve sticker information.

## Available Endpoints

### Create Guild Sticker

Create a new sticker for the guild.

**POST** `/guilds/:guild/stickers`

```typescript
import { createGuildSticker } from "@discordkit/client";

const sticker = await createGuildSticker({
  guild: "123456789",
  body: {
    name: "My Sticker",
    description: "A cool sticker",
    tags: "emoji, fun",
    file: /* PNG, APNG, or Lottie JSON file, max 500 KB */
  }
});
```

**Permissions:** Requires `MANAGE_GUILD_EXPRESSIONS` permission.

**Supports Audit Log Reason:** Yes

**File Requirements:**
- Must be PNG, APNG, or Lottie JSON file
- Maximum 500 KB
- Animated stickers constrained to 5 seconds
- Size constrained to 320 x 320 pixels

**Note:** Lottie stickers can only be uploaded on guilds with `VERIFIED` and/or `PARTNERED` features. Every guild has five free sticker slots by default, with more slots granted per Boost level.

**Returns:** The new sticker object on success. Fires a Guild Stickers Update Gateway event.

**Exports:**
- `createGuildSticker` - Handler function
- `createGuildStickerSchema` - Valibot schema for validation
- `createGuildStickerSafe` - Validated handler
- `createGuildStickerProcedure` - tRPC procedure (mutation)

### Delete Guild Sticker

**DELETE** `/guilds/:guild/stickers/:sticker`

**Permissions:** Requires `MANAGE_GUILD_EXPRESSIONS` permission.

**Supports Audit Log Reason:** Yes

**Returns:** `204 No Content` on success. Fires a Guild Stickers Update Gateway event.

**Exports:**
- `deleteGuildSticker` - Handler function
- `deleteGuildStickerSchema` - Valibot schema for validation
- `deleteGuildStickerSafe` - Validated handler
- `deleteGuildStickerProcedure` - tRPC procedure (mutation)

### Get Guild Sticker

Get a sticker from a guild.

**GET** `/guilds/:guild/stickers/:sticker`

```typescript
import { getGuildSticker } from "@discordkit/client";

const sticker = await getGuildSticker({
  guild: "123456789",
  sticker: "987654321"
});
```

**Returns:** A sticker object for the given guild and sticker IDs. Includes the `user` field if the bot has the `MANAGE_GUILD_EXPRESSIONS` permission.

**Exports:**
- `getGuildSticker` - Handler function
- `getGuildStickerSchema` - Valibot schema for validation
- `getGuildStickerSafe` - Validated handler
- `getGuildStickerProcedure` - tRPC procedure (query)
- `getGuildStickerQuery` - Query helper

### Modify Guild Sticker

Modify a guild sticker.

**PATCH** `/guilds/:guild/stickers/:sticker`

```typescript
import { modifyGuildSticker } from "@discordkit/client";

const sticker = await modifyGuildSticker({
  guild: "123456789",
  sticker: "987654321",
  body: {
    name: "Updated Name",
    description: "Updated description",
    tags: "updated, tags"
  }
});
```

**Permissions:** Requires `MANAGE_GUILD_EXPRESSIONS` permission.

**Supports Audit Log Reason:** Yes

**Note:** All parameters are optional.

**Returns:** The updated sticker object on success. Fires a Guild Stickers Update Gateway event.

**Exports:**
- `modifyGuildSticker` - Handler function
- `modifyGuildStickerSchema` - Valibot schema for validation
- `modifyGuildStickerSafe` - Validated handler
- `modifyGuildStickerProcedure` - tRPC procedure (mutation)
