## Running Examples Locally

All examples are available in the [DiscordKit repository](https://github.com/discordkit/discordkit) under the `examples/` directory.

```bash
cd examples/with-nextjs
npm install
cp .env.example .env.local
# Add your BOT_TOKEN to .env.local
npm run dev
```
